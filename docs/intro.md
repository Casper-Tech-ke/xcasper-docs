# 👻 Welcome to XCASPER Hosting

**XCASPER Hosting** is a production-ready game server hosting platform built on a heavily customized fork of [Pterodactyl v1.11](https://pterodactyl.io). It was designed for the African market and built to run **globally**.

---

## What Makes XCASPER Different?

| Feature | XCASPER | Vanilla Pterodactyl |
|---------|---------|---------------------|
| KES billing (Paystack) | ✅ Built-in | ❌ No |
| KES wallet with auto-renewal | ✅ | ❌ |
| Live CSS variable theming | ✅ No rebuild | ❌ |
| 9-tab Super Admin panel | ✅ | ❌ |
| Web push notifications (VAPID) | ✅ | ❌ |
| Email spam protection (6 layers) | ✅ | Basic only |
| Animated Casper ghost UI | ✅ | ❌ |
| One-click "Add Server" for users | ✅ | ❌ |

---

## Three Repositories

XCASPER is split across three focused repositories:

| Repo | Description |
|------|-------------|
| [xcasper-panel](https://github.com/Casper-Tech-ke/xcasper-panel) | The web control panel (Laravel + React) |
| [xcasper-wings](https://github.com/Casper-Tech-ke/xcasper-wings) | Wings daemon config + installation guide |
| [xcasper-eggs](https://github.com/Casper-Tech-ke/xcasper-eggs) | 18 ready-to-import server egg templates |
